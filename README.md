
# CIAC Registro

Aplicación de escritorio para Windows orientada al registro de participación de estudiantes en recepción CIAC.

## Objetivo
Registrar entradas y salidas de estudiantes por campus, almacenar los datos localmente y exportarlos a Excel.

## Flujo principal
1. Selección de campus
2. Ingreso de datos
3. Registro automático de entrada o salida según estado del alumno
4. Visualización de registros recientes
5. Exportación a Excel

## Requisitos del proyecto
- arquitectura modular
- base de datos local
- exportación a Excel
- validación de RUN y DV
- integración beta con escáner tipo teclado
- preparado para futuras actualizaciones
